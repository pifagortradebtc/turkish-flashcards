Данные, которые мы собираем

Приложение хранит данные только на вашем устройстве (локально): добавленные слова, прогресс обучения. Эти данные не передаются на наши серверы.

Приложение использует Google AdMob для показа рекламы. AdMob может собирать данные для персонализации объявлений в соответствии с политикой Google: https://policies.google.com/privacy
