# Настройка рекламы AdMob

## Реклама не обязательна

Реклама **не обязательна** для публикации в App Store. Это способ монетизации — вы зарабатываете на показах. Без рекламы приложение тоже можно публиковать.

---

## Что уже сделано

- Установлен плагин `@capacitor-community/admob`
- Добавлен баннер внизу экрана (показывается в нативном приложении)
- Межстраничная реклама показывается каждые 3 завершённые сессии карточек
- Используются **тестовые ID** — реклама показывается, но вы не зарабатываете

---

## Шаг 1: Регистрация в AdMob

1. Перейдите на https://admob.google.com/
2. Войдите через Google-аккаунт
3. **Приложения** → **Добавить приложение**
4. Выберите **iOS** и укажите название (например, Kelime Kartları)
5. Скопируйте **ID приложения** (формат: `ca-app-pub-XXXXXXXX~YYYYYYYY`)

---

## Шаг 2: Создание рекламных блоков

В AdMob Console:

1. **Медиация** или **Рекламные блоки** → **Создать рекламный блок**
2. **Баннер** — создайте блок, скопируйте ID (формат: `ca-app-pub-XXXXXXXX/YYYYYYYY`)
3. **Межстраничная** — создайте блок, скопируйте ID

---

## Шаг 3: Замена ID в коде

Откройте файл **`ads-init.js`** и замените:

```javascript
// Было (тестовые ID):
const BANNER_AD_ID = 'ca-app-pub-3940256099942544/6300978111';
const INTERSTITIAL_AD_ID = 'ca-app-pub-3940256099942544/1033173712';
const IS_TESTING = true;

// Стало (ваши ID):
const BANNER_AD_ID = 'ca-app-pub-ВАШ_ID/ВАШ_BANNER_ID';
const INTERSTITIAL_AD_ID = 'ca-app-pub-ВАШ_ID/ВАШ_INTERSTITIAL_ID';
const IS_TESTING = false;  // false при публикации!
```

---

## Шаг 4: Обновление iOS Info.plist

Откройте **`ios/App/App/info.plist`** и замените тестовый App ID на свой:

```xml
<key>GADApplicationIdentifier</key>
<string>ca-app-pub-ВАШ_ID~ВАШ_APP_ID</string>
```

Формат: `ca-app-pub-XXXXXXXX~YYYYYYYY` (с тильдой `~`, не слэшем).

---

## Шаг 5: Пересборка

После изменений выполните:

```powershell
npm run sync
```

На Mac: откройте Xcode и соберите заново.

---

## Важно

- **IS_TESTING: true** — показываются тестовые объявления, доход не идёт
- **IS_TESTING: false** — показываются реальные объявления, вы получаете доход
- Не используйте `IS_TESTING: false` с тестовыми ID — это нарушает правила AdMob
- Для GDPR (Европа) настройте UMP в AdMob Console

---

## Где показывается реклама

| Тип | Где |
|-----|-----|
| **Баннер** | Внизу экрана на всех страницах (index, spanish, english) |
| **Межстраничная** | Каждые 3 завершённые сессии карточек |

Изменить частоту межстраничной рекламы можно в `ads-init.js`:

```javascript
const INTERSTITIAL_EVERY_N_SESSIONS = 3;  // Например, 5 для реже
```
