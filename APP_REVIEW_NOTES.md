# Текст для поля «Примечания» в App Review Information (Guideline 2.1)

Apple требует заполнить раздел **App Review Information** перед повторной отправкой. Скопируйте текст ниже в поле «Примечания» (Notes).

---

## Текст для вставки (English — рекомендуется для Apple)

```
APP DESCRIPTION:
Flashcards app for learning Turkish, English and Spanish vocabulary. Users add their own words, study them in Learn mode and test knowledge in Quiz mode. Free app with ads (Google AdMob). No registration required.

ACCESS INSTRUCTIONS:
- App works without login. Launch the app and use immediately.
- Main flow: Tap "Add word" → enter word and translation → save. Then use "Learn" or "Quiz" mode.
- Switch languages via the language selector (Turkish/English/Spanish).

EXTERNAL SERVICES:
- Google AdMob: advertising (banner and interstitial ads). No other external services.

REGIONAL DIFFERENCES:
- App functions identically in all regions. No regional restrictions.

SCREEN RECORDING:
- Attached separately (see attachment). Shows: app launch, adding a word, Learn mode, Quiz mode, language switch.
```

## Важно: видеозапись экрана (Screen Recording)

Apple требует **видеозапись с реального устройства**, показывающую:
1. Запуск приложения
2. Добавление слова
3. Режим «Учу»
4. Режим «Проверка»
5. Переключение языков

**Как сделать:**
- iPhone: Настройки → Пункт управления → Запись экрана. Запустите приложение, запишите 1–2 минуты.
- Отправьте файл в **App Store Connect** → Проверка приложения → **Вложение** (Attachment).
